二、寫入grub4dos

grub4dos-0.4.5c
github

1.如上，從網絡上下載 grub4dos 到本機，並解壓，並為其全部文件添加完整權限。

2.cd到 grub4dos文件夾，插入U盤，將 grub4dos 寫入U盤。
（註意：是U盤掛載路徑/dev/sdc，不是分區掛載路徑/dev/sdc1）

安裝grub4dos到 U盤
